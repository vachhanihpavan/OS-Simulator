    Banker's Algorithm

    Banker's Algorithm is a resource allocation and deadlock avoidance algorithm that tests for safety by simulating the allocation of predetermined maximum possible amounts of all resources, and then makes an "safe-state" check to test for possible deadlock conditions for all other pending activities, before deciding whether allocation should be allowed to continue.

    Sockets

    A network socket is one endpoint in a communication flow between two programs running over the network. Socket programming allows processes to create logical sockets that communicate according to a predefined protocol over the network.

    File Management

    Time-Based OTP

    TOTP is two-factor authentication based on HMAC which combines a secret key with the current timestamp using a cryptographic hash function to generate a one-time password.

    Paging and Page Table

    Paging is a memory management scheme that eliminates the need for contiguous allocation of physical memory.
